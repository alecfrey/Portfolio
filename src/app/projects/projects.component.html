Today Aim - Accomplish Something Today

Today Aim is an iOS app that was created to help highlight all the tasks people accomplish in their daily lives. This application was created as a solo project. Today Aim was available to download on the App Store previously, but it is no longer supported. Brief Description:

  • Set one aim to accomplish at the start of each day
  • Record when the aim was accomplished
  • Sort collection by aims that you've accomplished or favorited
  • Calendar view allows for a bigger picture on the amount of aims created and accomplished
    • Technical Skills:
  • SwiftUI
  • Core Data Storage
  • Gained experience of whole development cycle from the idea to the deployment

    • Financio - Personal Finance Application

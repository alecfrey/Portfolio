Senior Design

Utilize Generative AI to Assess Student Learning

The Problem
  • Create faster personalized exams for class topics
  • Need a process that allows for educators to get more indepth reasoning behind students answers in exams
  • Create an automatic grading system to assess learning via AI chat bot
    • The Requirements
  • Based on given information of a topic, be able to “quiz” a student and assess his/her thought process
  • Implement the generative AI into Microsoft teams or Canvas
  • Maintain Ethical practices for student data and responses
  • Chat-GPT 4 since allows for file upload
  • Have high availability
    • Intended Users and Uses
  • Both students and professors will benefit from the Generative AI
  • Students will benefit in their studying from a personalized learning tool built specifically for the specific course
  • Professors will benefit through increased efficiency brought with the AI tools for grading purposes

    • It will also assist professors in being able to access true understandings of topics going past just right and wrong answers. As for how they will use it, professors will input class material into the AI, in which the AI will analyze and generate a series of questions to test the understanding of the students. Students will have an interactive exam, in which they will receive feedback and further questions to examine their understanding on the given class material.
